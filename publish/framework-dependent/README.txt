RAR Renamer v3.4.0 - Framework-Dependent Version
=================================================

This is the FRAMEWORK-DEPENDENT version (~1.6 MB).

REQUIREMENTS:
- 7-Zip installed (https://www.7-zip.org/)
- .NET 8 Runtime (https://dotnet.microsoft.com/download/dotnet/8.0)
- Windows 7/10/11

INSTALLATION:
1. Install 7-Zip
2. Install .NET 8 Runtime
3. Extract RarRenamer.exe
4. Run RarRenamer.exe

WINDOWS 7 USERS:
Also install Visual C++ Redistributable 2015-2022:
https://aka.ms/vs/17/release/vc_redist.x64.exe

For more information, visit:
https://github.com/L-at-nnes/RarRenamer.NET
