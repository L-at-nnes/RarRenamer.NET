RAR Renamer v3.4.0 - Self-Contained Version
===========================================

This is the SELF-CONTAINED version (~156 MB).

REQUIREMENTS:
- 7-Zip installed (https://www.7-zip.org/)
- Windows 7/10/11
- NO .NET Runtime required!

INSTALLATION:
1. Extract RarRenamer.exe
2. Install 7-Zip if not already installed
3. Run RarRenamer.exe

This version includes all .NET dependencies.

For more information, visit:
https://github.com/L-at-nnes/RarRenamer.NET
